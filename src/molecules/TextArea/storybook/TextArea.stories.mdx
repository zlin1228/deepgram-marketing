# TextArea

A textarea component for that grows vertically with content. The `TextArea` component automatically adjust the textarea
height on keyboard and window resize events.

## Usage

<br />

#### Label

The `label` prop takes a string that is output as the label above the Text area. Labels are required for screen readers
as per the WCAG

#### Helper Text

The `helperText` prop takes a string that is then output below the text area. The helper text is used to help instruct
the user on how to interact with the UI and to deliver error messages upon validation failures. Helper text is red if
`error == true`.

#### Controlled

The `TextArea` is controlled component that expects a state driven `value` from the components `onChange` prop.

```jsx
<TextArea value={value} onChange={({ target }) => setValue(target.value)} />
```
