# Checkboxes

Checkboxes allow users to select one or more items from a set, while radio buttons allow users to select just one option
from a set. Both can also be used to turn an option on or off.

## Usage

### Controlled component

You can control the checkbox with the `value` and `onChange` props

#### Label & helper text

The `label` and `helperText` props are both optional and inherently light/dark mode optimized.
