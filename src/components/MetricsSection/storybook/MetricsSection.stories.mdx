# Metrics Section

Metrics sections are useful for communicating authority and social proof to users.

## Usage

#### Variant

The `variant` prop is required and determines what variant the component is

#### Heading

The `heading` prop is required. The `headingTextColor` prop is optional and determines the text color of `heading`

#### Description

The `description` prop is required. The `descriptionTextColor` prop is optional and determines the text color of
`description`

#### Icon

The `icon` prop is optional and determines the icon to show above `heading`

#### Card Background Color

The `cardBackgroundColor` prop is optional and determines the background color of the `card` variant

#### Accent Line Color

The `accentLineColor` prop is optional and determines the accent line color of the `centered` and `accent-line` variants

### Metric Props

The `metricHeadingTextColor` prop is optional and determines the color of the metric value

The `metricDescriptionTextColor` prop is optional and determines the color of the metric description in the `action-1`
and `action-2` variants

The `metricCtaTextColor` prop is optional and determines the color of the metric CTA in the `accent-line` variant
