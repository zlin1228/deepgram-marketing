# Quote

The quote is a full width testimonial style quote that includes optional `avatar`, `company`, `name`, `rating`, and
`role`. Any fields may be omitted to create any variation of quotes.

### Rating

The ratings takes a number 1-5 by .1 increments and shows correct stars accordingly.
